
La méthode toString permet de retourner la représentation d'un objet en format textuel.

La méthode equals est utilisée pour établir un lien d'égalité entre deux objets identiques. 
Elle doit être utilisée avec hashCode car elle permet de vérifier que les deux objets ont le même hachage.
Appeler la méthode hashCode sur deux objets qui valident la méthode equals. 

La méthode hashCode est utilisée pour calculer le hachage d'une clef. 
Concrètement, la méthode retourne un nombre entier généré par un algorithme.

Dans notre exercice, cela permet de vérifier le lien d'égalité entre deux idAnimal.
